# Pairwise products

## Description:

Write a program that finds the product of pairs of numbers in a list. A pair is considered to be the first and the last element, the second and the second-to-last, and so on.

## Example:

```
Input: [2, 3, 4, 5, 6]
Output: [12, 15, 16]
```
```
Input: [2, 3, 5, 6]
Output: [12, 15]
```
